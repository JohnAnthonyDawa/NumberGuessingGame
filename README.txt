Number Guessing Game

**Brief Description of the Game**
The Number Guessing Game is a simple and interactive browser-based game where players try to guess a randomly generated number. The game offers three difficulty levels (Easy, Medium, Hard), each with a different range of numbers to guess from. The goal is to correctly guess the number in as few attempts as possible.

**Instructions on How to Play**
Select the difficulty level from the dropdown menu:
**Easy**: Guess a number between 1 and 100.
**Medium**: Guess a number between 1 and 250.
**Hard**: Guess a number between 1 and 500.
Enter your guess into the input field provided.
Click the "Submit Guess" button to check your guess.
**The game will provide feedback:**
"Too low!" if your guess is lower than the random number.
"Too high!" if your guess is higher than the random number.
"Congratulations!" if you guess the correct number.
If you guess the correct number, you can restart the game by clicking the "Restart Game" button.
**Special Features or Functionalities**
**Difficulty Levels**: Players can choose between Easy, Medium, and Hard, which affects the range of the random number.
**Real-Time Feedback**: Feedback is provided after each guess, helping players adjust their next attempt.
Automatic Restart on Difficulty Change: Changing the difficulty level resets the game automatically.
**Interactive UI**: A clean, responsive, and visually appealing user interface with smooth transitions.
